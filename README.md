XYplotter.
===============================

Drawing graphic images on paper.

Can be used as typographic builder or Bezier's curves builder.


---
### Requirements

  * Python 3.4+
  * Pillow

### Installation

```bash
git clone https://github.com/Alexponomarev7/plotter
cd XYplotter
python3 setup.py install
```
